## 🚀 快速使用

1. 根据当前测试阶段，进入对应分类目录。
2. 每个能力项目下包含：
   - `SKILL.md` / `SKILLS.md`：能力核心描述与执行指南。
   - `scripts/`：可执行代码或利用工具。
   - `references/`：参考资料与能力清单。
   - `assets/`：图片、字体、输出示例等。
3. 直接阅读能力文档，或执行工具（需在安全测试环境中使用）。

## 🔧 构建方式

本项目由 AI 自动化流水线从**30+ 个开源安全能力项目**中提取、分析、分类并重组织而成。

### 流程概述

1. **能力提取**：扫描源仓库，识别标准化的能力单元（`SKILL.md` 或单文件能力）。
2. **功能分类**：使用大语言模型将每个能力归入精细化的红队攻击阶段分类体系。
3. **本地组织**：按分类创建层级目录，完整保留原始能力的所有文件（包括源码、参考资料、资源）。

详细构建原理参见 [CLASSIFICATION_SUMMARY.md](./CLASSIFICATION_SUMMARY.md)。
