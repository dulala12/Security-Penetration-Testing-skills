## 🚀 快速使用

1. 根据当前测试阶段，进入对应分类目录。
2. 每个技能目录下包含：
   - `SKILL.md` / `SKILLS.md` ：技能核心描述与执行步骤。
   - `scripts/` ：可执行代码或利用脚本。
   - `references/` ：参考资料与技术背景。
   - `assets/` ：模板、字典、输出示例等。
3. 直接阅读技能文档，或执行脚本（需在授权测试环境中使用）。

## 🤖 构建方式

本项目由 AI 自动化流水线从 **30+ 个开源安全技能仓库** 中提取、分析、分类并重组而成。

### 流程简介

1. **技能提取**：扫描源仓库，识别标准化的技能单元（`SKILL.md` 或单文件技能）。
2. **功能分类**：使用大语言模型将每个技能归入精细化的红队攻击阶段分类体系。
3. **本地组织**：按分类创建层级目录，完整保留原始技能的所有文件（包括脚本、参考资料、资产）。

详细构建原理见 [CLASSIFICATION_SUMMARY.md](./CLASSIFICATION_SUMMARY.md)。
