# GDPR/CCPA 合规检查清单

## 数据收集

- [ ] 是否明确告知用户收集了哪些数据
- [ ] 数据收集是否有合法依据（用户同意/合同必需/合法利益）
- [ ] 是否遵循数据最小化原则（仅收集必需数据）
- [ ] 是否收集了敏感个人数据（健康、生物识别、种族等）

## 数据使用

- [ ] 数据使用目的是否明确且与声明一致
- [ ] 是否将数据用于声明之外的目的
- [ ] 是否将数据分享给第三方
- [ ] 第三方数据处理是否有合同保障

## 用户权利

- [ ] 用户是否可以查看被收集的数据（访问权）
- [ ] 用户是否可以要求删除数据（被遗忘权）
- [ ] 用户是否可以导出自己的数据（可携带权）
- [ ] 用户是否可以撤回同意

## 数据安全

- [ ] 传输是否加密（TLS 1.2+）
- [ ] 存储是否加密
- [ ] 是否有访问控制机制
- [ ] 是否有数据泄露应急响应计划

## Skill 特定检查

- [ ] Skill 是否在 SKILL.md 中声明了数据收集行为
- [ ] 是否静默收集用户未预期的数据
- [ ] 环境变量/文件内容的读取是否必要且已声明
- [ ] 外发网络请求是否传输了用户个人数据
