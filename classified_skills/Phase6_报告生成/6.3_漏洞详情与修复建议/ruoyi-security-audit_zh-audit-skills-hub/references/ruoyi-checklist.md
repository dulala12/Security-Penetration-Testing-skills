# 若依（RuoYi）审计清单

## 1) 认证与授权
- 登录接口是否存在绕过路径。
- `@PreAuthorize` / 数据权限注解是否覆盖关键接口。
- 角色与菜单权限是否与后端接口强绑定。

## 2) 数据权限与越权
- 是否仅校验登录而未校验资源归属。
- 是否存在用户可控 `deptId/userId/roleId` 导致越权。
- 导出、详情、删除、批量操作是否逐条鉴权。

## 3) SQL 注入
- 动态排序字段是否白名单。
- 查询条件是否拼接 SQL。
- Mapper/XML 中 `${}` 使用是否受控。

## 4) 文件上传下载
- 上传是否校验扩展名+MIME+魔数。
- 上传目录是否可执行。
- 下载接口路径是否可控并可穿越。

## 5) 定时任务与系统管理
- 定时任务配置是否可被低权限用户修改。
- 任务执行目标是否可控。
- 系统参数接口是否存在未授权修改。

## 6) 配置与信息泄露
- 默认口令/弱口令。
- 敏感配置（数据库、token）是否泄露。
- 错误堆栈与调试信息是否对外。
